# ЭлектроЦентр — Прототип Smart TV

Прототип цифрового эфира для Smart TV в магазине ЭлектроЦентр.

## Просмотр

Откройте: **https://kokolsk1y.github.io/electrocentre-tv-proto/**

## Что внутри

- 15 слайдов в ротации (инфо, баннеры, AWS-бренд, допродажа)
- 3 типа переходов: Diagonal Wipe, Zoom Morph, Circuit Trace
- Новая система появления контента
- 2 новых AWS-слайда (Aura, Uno)
- 2 слайда допродажи («Не забыл?», «Собери щит»)

## Управление

- **Стрелки влево/вправо** — листать слайды вручную
- **Клик** — следующий слайд
- Автоматическое переключение работает по умолчанию

## Статус

Это **прототип**, не production. Production-версия работает на https://kokolsk1y.github.io/electrocentre-tv/
